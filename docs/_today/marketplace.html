What's new in 1.0.3

Full changelog
  • ✨New Session Activity Dashboard — live rollup of current focus, active work, deploy pushes, CI status, and open issues
  • 🛠️Marketplace "What's inside" now generated from the command catalog (121 commands, 13 groups) instead of 4 hardcoded cards
  • 🛠️Preview/Start button now cache-busts every launch so edits show up without a manual hard-reload
  • 🛠️BYOK main model no longer errors on Copilot's internal utility-only models
  • 🛠️Marketplace Compliance now catches stale (not just missing) CHANGELOG.md files
  • 🐛Fixed false-positive regression failures and health-state save errors from bg-health-runner
  • 🐛Fixed a silent MCP server crash that discarded its own error message on exit
  • 🐛Fixed untagged code-fence warnings across project docs
