Getting Started

Get Cielo Vista Tools installed and running in VS Code in under two minutes.

1

Download the VSIX

All Cielo Vista tools are bundled in a single extension file.

download Download cielovista-tools-1.0.2.vsix
2

Install in VS Code

Install the downloaded file directly into VS Code Insiders.

  1. chevron_right Open VS Code Insiders
  2. chevron_right Open the Extensions panel — Ctrl+Shift+X
  3. chevron_right Click the ⋯ menu (top right of the panel) → Install from VSIX…
  4. chevron_right Select cielovista-tools-1.0.2.vsix and click Install
  5. chevron_right Reload VS Code when prompted
3

Verify the Install

Confirm everything is working in the Command Palette.

  1. chevron_right Press Ctrl+Shift+P to open the Command Palette
  2. chevron_right Type CVS — you should see all Cielo Vista commands listed
