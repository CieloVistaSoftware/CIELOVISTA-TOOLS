# Image Reader Feature

## Overview
This feature provides an image reader webview panel using HTML, CSS, and JS assets migrated from CodePilot-Wake-Monitor.

## Commands
- `cvs.imageReader.open`: Opens the Image Reader panel.

## Usage
1. Run the `CieloVista: Open Image Reader` command from the Command Palette.
2. The panel will display the image reader UI.

## Assets
- HTML, CSS, and JS are located in `src/features/image-reader-assets/`.

## Test Steps
- Open the command and verify the panel loads and functions as expected.
